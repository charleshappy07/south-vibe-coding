# South Vibe Coding Club

**Website:** [southvibecoding.com](https://southvibecoding.com)

A club website for **South Vibe Coding Club** — a free after-school program at South Plantation High School where students build real projects using AI-powered coding tools.

---

## What is Vibe Coding?

Vibe Coding is a style of programming where you describe what you want in plain language and AI tools generate the code. Students focus on **creativity and problem-solving** rather than memorizing syntax — turning ideas into working websites, games, and apps from day one.

## What Members Build

| Project Type | Description | Tools |
|---|---|---|
| Interactive Websites | Personal portfolios, landing pages | Lovable, React, Tailwind |
| Mini-Games | Browser games with scoring and animations | p5.js, JavaScript |
| Utility Apps | Grade calculators, planners, productivity tools | React, APIs |
| Interactive Stories | Choose-your-own-adventure narratives | Twine, Flute |

---

## 10-Week Curriculum

| Week | Topic | Deliverable |
|---|---|---|
| 1 | The Vibe of Coding | First prototype website via Lovable |
| 2 | Words → Web | Personal portfolio page live |
| 3 | Interactive Art | p5.js sketch reacting to mouse/keys |
| 4 | Make a Mini-Game | Playable JS mini-game with score/lives |
| 5 | Story & Choices | Interactive story with branches |
| 6 | Build a Tool | Utility app (grade calculator, planner) |
| 7 | Level Up with AI IDEs | Improved features and bug fixes |
| 8 | Project Sprint I | Final project started |
| 9 | Project Sprint II | Final project polished |
| 10 | Demo Day | Presented to the club |
